## 추후 업데이트할 기능
* 총 만들어진 도메인 수 보기
* 복사 붙혀넣기가 쉽도록 input 박스 제공
* 3글자 이상부터는 유저들이 직접 URL을 만들 수 있도록
* API 제공
* 2글자 도메인 바닥났을때를 대비 (먼산) - 10번 시도 후 안되면 오래된 것을 지우고 그자리를 꿰찬다.
* 정기적으로 도메인 연결 제거 (1년간 접속을 하지 않은 URL)
* 크롬 확장 프로그램
